# Cesta React Native
Projeto inicial de desenvolvimento mobile com react-native

### Como executar o projeto ?
 ##### Projeto feito com expo
 
 * Baixe e instale o Node 12 LTS ou superio para usar o Expo CLI
 * npm install -g expo-cli
 * cd cesta-react-native
 * npm start
 * Baixe o aplicativo Expo Go no seu smarth fone 
 * Escaneie o QRCode que irá aparecer na tela do seu terminal
 * Ou caso já tenha o Android Estudio instalado e configurado basta apertar a tecla A do seu teclado para chamar a virtualização
